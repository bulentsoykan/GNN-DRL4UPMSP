Abstract

This paper addresses the Unrelated Parallel Machine Scheduling Problem (UPMSP) with release dates, sequence-dependent and machine-dependent setup times, and machine eligibility constraints. We propose a novel Deep Reinforcement Learning (DRL) framework that combines Proximal Policy Optimization (PPO) with Heterogeneous Graph Neural Networks (GNN) to simultaneously minimize Total Weighted Tardiness (TWT) and Total Setup Time (TST). The heterogeneous GNN captures the complex relationships between jobs, machines, and setup configurations through attention-based message passing. Experimental results on benchmark instances demonstrate that our approach significantly outperforms a standard dispatching rule (ATCSR_Rm) and a metaheuristic (Genetic Algorithm), providing a robust and scalable solution for complex manufacturing scheduling.

Method Overview

We formulate the UPMSP as a Markov Decision Process and solve it with a PPO agent equipped with a heterogeneous Graph Neural Network for state representation. The GNN encodes jobs, machines, and setup types as distinct node types, connected by edges representing eligibility, processing times, and setup costs.

Heterogeneous GNN 4-layer GATv2 with 128-dim embeddings and 4 attention heads. Captures multi-relational structure between jobs, machines, and setups.
PPO Agent Actor-critic architecture with shared GNN backbone. Action masking ensures feasible job-machine assignments.
Multi-Objective Reward Reward r_t = -α·ΔTWT - β·setup_time balances conflicting scheduling objectives.
Real-World Constraints Handles release dates, machine eligibility restrictions, and sequence/machine-dependent setup times.

Problem Description

The Unrelated Parallel Machine Scheduling Problem (UPMSP) involves assigning n jobs to m unrelated machines to optimize scheduling objectives. Each job has machine-dependent processing times, a release date, a due date, and a weight reflecting its priority. Setup times are both sequence-dependent and machine-dependent, and each job can only be processed on a subset of eligible machines.

This NP-hard combinatorial optimization problem models real manufacturing scenarios in semiconductors, textiles, and other industries. Traditional methods (heuristics, metaheuristics, exact solvers) struggle with the scale and multi-objective nature of realistic instances.
